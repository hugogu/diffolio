Dictionary Management Overview

Dictionary management is the core module of the platform, providing full lifecycle management including dictionary creation, version management, and file upload. Through an intuitive interface and powerful features, you can easily manage various types of dictionary resources.

Supported File Formats

.txt

Text File

Plain text format, recommended for dictionary content

  • UTF-8 encoding
  • Clear structure
  • Wide compatibility
  • Recommended
.doc

Word 97-2003

Legacy Word format

  • Legacy Office compatible
  • Supports rich text
  • Common format
  • Auto parsing
.docx

Word 2007+

Modern Word format, recommended for rich text documents

  • New Office standard
  • Supports complex formatting
  • Compressed storage
  • Accurate parsing
.pdf

PDF Document

Portable Document Format

  • Fixed format
  • Cross-platform compatible
  • Suitable for scanned documents
  • OCR parsing
