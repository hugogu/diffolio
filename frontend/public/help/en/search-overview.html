Search Tips

⭐

Use Quotes for Exact Match

Use double quotes around search terms for exact match, avoiding fuzzy search results.

"computer network"
Only matches entries with exact same content
⭐

Use Wildcards

Use * to match any characters, expanding search scope.

network*
Matches "network", "network technology", "network system", etc.
⭐

Combine Search Conditions

Use logical operators like AND, OR, NOT to combine multiple search conditions.

network AND computer
Results containing both "network" and "computer"
⭐

Limit Search Scope

Use filters to limit search scope by dictionary, version, or part-of-speech.

life + POS:verb
Search "life" within verbs

FAQ

What to do if no results are found?
Try: 1) Check if spelling is correct; 2) Use more general keywords; 3) Clear filter conditions; 4) Try using wildcards.
What to do if there are too many search results?
You can: 1) Use exact match (double quotes); 2) Add more keywords; 3) Use filters to narrow scope; 4) Use more specific search terms.
How to search a specific dictionary?
Select target dictionary in the filter, or set dictionary scope in advanced search.
Does search support pinyin?
Yes, the system supports pinyin search. You can input pinyin to find corresponding Chinese character entries.
