File Upload

The file upload function supports multiple dictionary formats. Through intelligent parsing configuration, dictionary content is converted into structured data, providing a foundation for subsequent version comparison and analysis.

Supported File Formats

TXT

Text File

Plain text format, suitable for dictionaries with simple structure

  • UTF-8 encoding recommended
  • Supports line breaks
  • Easy to edit and view
  • Moderate file size
JSON

JSON Format

Structured data, suitable for program processing

  • Standard data format
  • Clear hierarchical structure
  • Supports complex data types
  • Easy for program parsing
XML

XML Format

Markup language, suitable for complex dictionaries

  • Self-describing structure
  • Supports namespaces
  • Stylesheet support
  • Internationalization friendly
XLSX

Excel Spreadsheet

Spreadsheet format, suitable for batch processing

  • Visual editing
  • Formula support
  • Data validation features
  • Widely compatible format

Parsing Configuration

Parsing configuration defines how the system recognizes and extracts entry information from dictionaries:

Required
Entry Pattern - Defines entry recognition rules, such as regular expressions
Required
Definition Pattern - Defines extraction rules for definition content
Optional
Example Pattern - Defines recognition rules for example sentences
Optional
POS Pattern - Defines extraction rules for part-of-speech information

File Requirements

File Size

  • Maximum single file size 100MB
  • Recommended file size under 10MB
  • Large files may take longer to process
  • Please split files if over limit

Encoding Format

  • UTF-8 encoding recommended
  • Avoid using old encodings like GBK, GB2312
  • Check file encoding before upload
  • Encoding errors may cause parsing failures

File Structure

  • Ensure clear and consistent file structure
  • Avoid using complex formatting
  • Check for special characters
  • Recommended to test with small batches first

FAQ

What to do if upload fails?
Check network connection, file size, and if format meets requirements. You can try re-uploading or contact technical support.
How to improve parsing accuracy?
Optimize parsing configuration, use suitable recognition rules, and ensure clear and consistent file structure.
Can I upload multiple files at once?
Currently supports single file upload. For multiple files, please upload separately or contact technical support.
