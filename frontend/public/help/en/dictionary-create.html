1

Optional
Publisher - Enter the publisher name, e.g., "Commercial Press"
Required
Language - Select the main language of the dictionary, default is Chinese
Optional
Description - Enter a brief description and characteristics of the dictionary

3

Click the Create button to complete creation

4

Field Filling Recommendations

Required
Dictionary Name - As the main identifier, it is recommended to use the full official name for easy retrieval and distinction.
Optional
Publisher - Recommended to fill in for subsequent filtering and archiving.
Required
Language - Affects search and parsing strategies, please keep consistent with the source file language.
Optional
Description - Record scope of application and version background for team members to quickly understand.

Notes

Naming Conventions

  • Use the official full dictionary name, avoid abbreviations
  • If version information is included, please indicate it in the name
  • Avoid using special characters and emojis
  • Recommended length between 2-50 characters

Information Completeness

  • Fill in complete publisher information whenever possible
  • Description can include the target audience of the dictionary
  • If it is a series dictionary, please explain in the description
  • Accurately select language type

FAQ

Can the dictionary name be changed?
After dictionary creation, the name can be modified, but it is recommended to use an accurate name during creation to avoid frequent changes later.
What languages are supported?
Currently supports Chinese, English, Japanese, Korean, and other languages. The system will provide corresponding parsing configurations based on language selection.
Can it be deleted after creation?
Yes, it can be deleted, but if the dictionary already has version data, all versions must be deleted first before the dictionary itself can be deleted.
