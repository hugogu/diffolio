FAQ

What dictionary formats are supported?
The system supports various dictionary formats, including TXT, JSON, XML, Excel, and other common formats. For specific format requirements, please refer to the file upload help documentation.
How can I improve parsing accuracy?
You can create custom parsing configurations and set parsing rules suitable for your dictionary format. The system provides configuration templates and examples for reference.
Can comparison results be exported?
The current help documentation focuses on comparison viewing and analysis workflows. You can view complete results on the version comparison page.
