分类词典概述

分类词典功能支持用户上传自定义分类体系，建立词汇与分类的映射关系，用于辞书研究中的语义分析和主题检索。平台不内置固定分类，由用户根据研究需求自行定义和上传。

分类体系结构

分类体系采用层次化结构，支持多级分类：

L1

一级分类

最高级别的分类，如"科技"、"文化"

  • 每个一级分类包含多个二级分类
  • 定义分类的基本范畴
  • 提供分类的整体框架
L2

二级分类

中间级别的分类，如"计算机技术"、"文学艺术"

  • 细化一级分类的内容
  • 连接具体词汇和高级分类
  • 提供更精确的分类定位
L3

三级分类

具体的分类类别，如"编程语言"、"古典文学"

  • 最细粒度的分类级别
  • 直接关联具体词汇
  • 支持精确的语义定位

数据格式

CSV

CSV格式

逗号分隔值格式，适合批量数据处理

  • 词汇,分类1,分类2,分类3
  • 支持Excel直接编辑
  • 易于批量导入
  • 格式简单直观
JSON

JSON格式

结构化数据格式，适合程序处理

  • 层次化数据结构
  • 支持复杂映射关系
  • 便于API集成
  • 支持元数据
XML

XML格式

标记语言格式，适合复杂分类体系

  • 自描述结构
  • 支持属性和命名空间
  • 样式表支持
  • 国际化友好

常见问题

如何创建自定义分类体系？
可以在分类管理页面创建新的分类体系，定义分类结构和映射关系。
支持多少级分类层次？
系统支持多级分类，建议不超过5级，以免过于复杂影响使用效果。
分类数据可以导出吗？
当前版本支持分类上传，导出功能计划在后续版本中提供。
